━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第一步：手动下载 Python
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【方式 1】从官网下载（推荐）

1. 打开浏览器，访问：
   https://www.python.org/downloads/

2. 点击 "Download Python 3.11.x" 按钮

3. 或者直接下载：
   https://www.python.org/ftp/python/3.11.8/python-3.11.8-amd64.exe

4. 下载完成后，将文件重命名为：
   python_installer.exe

5. 放在项目根目录（与 main.py 同一目录）

【方式 2】从国内镜像下载（更快）

1. 访问华为云镜像：
   https://repo.huaweicloud.com/python/

2. 找到 3.11.8 目录，下载：
   python-3.11.8-amd64.exe

3. 重命名为：python_installer.exe

4. 放在项目根目录

【方式 3】从淘宝镜像下载

1. 访问：
   https://registry.npmmirror.com/-/binary/python/

2. 找到 3.11.8 目录

3. 下载 python-3.11.8-amd64.exe

4. 重命名为：python_installer.exe

5. 放在项目根目录

双击运行：手动安装Python并打包.bat

脚本会：
1. 检测 python_installer.exe 是否存在
2. 安装 Python 到 python_local 目录
3. 安装依赖
4. 打包生成 exe

Q: 下载很慢怎么办？
A: 使用国内镜像（华为云或淘宝镜像）

Q: 下载失败怎么办？
A: 换一个镜像源，或使用下载工具（如迅雷、IDM）

Q: 文件名必须是 python_installer.exe 吗？
A: 是的，脚本会查找这个文件名

Q: 可以用其他版本的 Python 吗？
A: 可以，但推荐 3.11.x 或 3.10.x

Q: 下载后可以删除安装程序吗？
A: 可以，打包完成后会自动删除
