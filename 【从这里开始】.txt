【方案 2】手动下载 Python 本地打包

  第一步：手动下载 Python
  • 访问华为云镜像（国内快）
  • 下载 python-3.11.8-amd64.exe
  • 重命名为 python_installer.exe
  • 放在项目根目录
  
  第二步：运行打包脚本
  • 双击：手动安装Python并打包.bat
  
  详见：手动下载Python方案.txt

【方案 3】传统方式（需要安装 Python）

  1. 查看：安装Python指南.txt
  2. 安装 Python 3.10 或 3.11
  3. 双击：build.bat
