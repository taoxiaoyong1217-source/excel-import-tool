【最简单的方法】

  双击运行：自动安装并打包.bat
  
  脚本会自动完成：
  ✓ 检测系统 Python（如果有就用系统的）
  ✓ 如果没有，自动下载 Python 便携版（约 30MB）
  ✓ 配置 Python 环境
  ✓ 安装 pip
  ✓ 安装所有依赖
  ✓ 打包生成 exe
  
  全程自动，无需任何手动操作！

【生成的文件】

  dist\Excel导入工具.exe
  
  • 完全独立的可执行文件
  • 约 30-50 MB
  • 可在任何 Windows 电脑运行
  • 无需 Python 环境

【便携版 Python】

  如果系统没有 Python，脚本会下载到：
  python_portable\ 目录
  
  • 这是一个便携版 Python
  • 只用于打包，不影响系统
  • 可以随时删除（打包完成后）
  • 下次打包会重复使用

【网络要求】

  首次运行需要下载：
  • Python 便携版（约 30 MB）
  • pip 安装脚本（约 2 MB）
  • Python 依赖包（约 50 MB）
  
  总计约 80-100 MB
  后续打包无需重新下载

【时间估计】

  • 首次运行：5-10 分钟（需要下载）
  • 后续打包：1-3 分钟

【常见问题】

  Q: 下载失败？
  A: 检查网络连接，或使用 VPN
  
  Q: 打包失败？
  A: 查看错误信息，可能是依赖安装问题
  
  Q: 可以删除 python_portable 吗？
  A: 打包完成后可以删除，下次会重新下载
  
  Q: 为什么这么慢？
  A: 首次需要下载 Python 和依赖，后续会快很多

【手动清理】

  如果想重新开始：
  1. 删除 python_portable 目录
  2. 删除 build 和 dist 目录
  3. 重新运行脚本
