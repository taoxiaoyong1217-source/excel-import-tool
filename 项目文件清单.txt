========================================
Excel 导入工具 - 项目文件清单
========================================

【核心代码文件】
├── main.py                  # 程序入口
├── main_window.py           # 主窗口界面
├── config_window.py         # 环境配置窗口
├── config_manager.py        # 配置管理模块
└── api_client.py            # API 客户端模块

【配置文件】
├── config.json              # API 配置文件（自动生成）
└── requirements.txt         # Python 依赖列表

【打包相关】
├── build.spec               # PyInstaller 配置文件
├── build.bat                # 自动打包脚本（推荐）
├── build_simple.bat         # 简化打包脚本
└── test_run.bat             # 测试运行脚本

【文档文件】
├── README.md                # 项目说明（开发者）
├── build.md                 # 打包详细说明
├── 使用说明.txt             # 用户使用说明
├── 打包步骤.txt             # 快速打包指南
└── 项目文件清单.txt         # 本文件

【其他】
└── .gitignore               # Git 忽略文件

【打包后生成】
├── build/                   # 临时构建文件（可删除）
└── dist/
    └── Excel导入工具.exe    # 最终的独立可执行文件

========================================
快速开始
========================================

【开发测试】
1. 双击 test_run.bat 运行程序

【打包为 EXE】
1. 双击 build.bat 自动打包
2. 生成的 exe 在 dist 目录

【分发给用户】
只需要分发 dist\Excel导入工具.exe 文件！
