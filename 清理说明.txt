【快速清理】

  双击运行：清理Python环境.bat
  
  脚本会自动删除：
  • Python 便携版（python_portable 目录）
  • 打包临时文件（build 目录）
  • 打包输出目录（dist 目录，可选）

【Python 便携版】

位置：python_portable\ 目录
大小：约 100-150 MB
用途：仅用于打包，不影响系统

删除方式：
• 方式1：运行 清理Python环境.bat
• 方式2：直接删除 python_portable 文件夹

影响：
✓ 不影响系统 Python（如果有）
✓ 不影响已生成的 exe 文件
✓ 下次打包会重新下载

【打包临时文件】

位置：build\ 目录
大小：约 50-100 MB
用途：PyInstaller 打包过程的临时文件

删除方式：
• 方式1：运行 清理Python环境.bat
• 方式2：直接删除 build 文件夹

影响：
✓ 可以安全删除
✓ 下次打包会重新生成

【打包输出目录】

位置：dist\ 目录
大小：约 30-50 MB
内容：Excel导入工具.exe

删除方式：
• 方式1：运行 清理Python环境.bat（会询问）
• 方式2：直接删除 dist 文件夹

注意：
⚠️  删除前请先备份 Excel导入工具.exe
⚠️  这是最终的可执行文件

【打包完成后立即清理】

如果你：
✓ 已经获得了 Excel导入工具.exe
✓ 已经复制到其他位置
✓ 近期不会再次打包

可以立即清理，释放约 200 MB 空间

【保留 Python 环境】

如果你：
✓ 可能需要修改代码后重新打包
✓ 想避免重新下载 Python
✓ 磁盘空间充足

可以保留 python_portable 目录

【部分清理】

可以只删除：
• build 目录（临时文件，可以随时删除）
• dist 目录（备份 exe 后删除）

保留：
• python_portable 目录（下次打包更快）

Q: 删除后还能打包吗？
A: 可以，重新运行打包脚本会自动下载 Python

Q: 会影响系统 Python 吗？
A: 不会，便携版 Python 完全独立

Q: 删除后 exe 还能用吗？
A: 可以，exe 是独立的，不依赖 Python 环境

Q: 需要删除源代码吗？
A: 不需要，源代码保留，方便以后修改

Q: 删除失败怎么办？
A: 关闭所有相关程序，或重启电脑后再试

Q: 可以只删除 Python 吗？
A: 可以，只删除 python_portable 目录即可

【方案 A：完全清理】
适用：打包完成，不再需要

1. 复制 dist\Excel导入工具.exe 到其他位置
2. 运行 清理Python环境.bat
3. 全部删除

释放空间：约 200 MB

【方案 B：保留 Python】
适用：可能还会打包

1. 只删除 build 和 dist 目录
2. 保留 python_portable 目录

释放空间：约 100 MB

【方案 C：最小清理】
适用：磁盘空间充足

1. 只删除 build 目录
2. 保留其他所有文件

释放空间：约 50 MB
