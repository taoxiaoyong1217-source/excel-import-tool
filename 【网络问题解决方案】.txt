如果遇到 "下载失败" 或 "连接被关闭" 错误，请使用以下方案：

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
方案 1：手动下载（强烈推荐）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【步骤 1】使用浏览器下载 Python

打开浏览器，访问以下任一地址：

1. 华为云镜像（推荐，国内最快）：
   https://repo.huaweicloud.com/python/3.11.8/python-3.11.8-amd64.exe

2. 淘宝镜像：
   https://registry.npmmirror.com/-/binary/python/3.11.8/python-3.11.8-amd64.exe

3. 清华镜像：
   https://mirrors.tuna.tsinghua.edu.cn/python/3.11.8/python-3.11.8-amd64.exe

4. 官方（国外）：
   https://www.python.org/ftp/python/3.11.8/python-3.11.8-amd64.exe

【步骤 2】重命名文件

下载完成后，将文件重命名为：
python_installer.exe

【步骤 3】放在项目目录

将 python_installer.exe 放在项目根目录
（与 main.py、build.bat 等文件在同一目录）

【步骤 4】运行打包脚本

双击运行：手动安装Python并打包.bat

完成！

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
方案 2：使用下载工具
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

如果浏览器下载也失败，使用专业下载工具：

1. 迅雷
2. Internet Download Manager (IDM)
3. Free Download Manager

这些工具支持：
✓ 断点续传
✓ 多线程下载
✓ 更稳定

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
方案 3：使用系统自带 Python（如果有）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

检查系统是否已安装 Python：

1. 按 Win + R
2. 输入 cmd 回车
3. 输入 python --version

如果显示版本号，说明已安装 Python，直接运行：
build.bat

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
方案 4：从 Microsoft Store 安装
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Windows 10/11 用户可以：

1. 打开 Microsoft Store
2. 搜索 "Python 3.11"
3. 点击安装
4. 安装完成后运行：build.bat

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
方案 5：使用其他电脑
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

如果有其他电脑（公司、朋友、网吧等）：

1. 将项目文件夹复制到那台电脑
2. 在那台电脑运行打包脚本
3. 将生成的 Excel导入工具.exe 复制回来

这个 exe 可以在任何 Windows 电脑运行！

为什么会下载失败？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

常见原因：
1. 网络不稳定
2. 防火墙拦截
3. 代理设置问题
4. 服务器限制
5. 文件太大（约 30 MB）

解决方法：
• 使用国内镜像源
• 使用浏览器下载（支持断点续传）
• 使用下载工具
• 换个网络环境
