【问题】
运行 build.bat 提示"未找到 Python"

【原因】
你的电脑还没有安装 Python，或者 Python 没有添加到系统环境变量

【解决方案】

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
方法一：安装 Python（推荐）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

步骤 1：下载 Python
  访问官网：https://www.python.org/downloads/
  
  或直接下载：
  https://www.python.org/ftp/python/3.11.8/python-3.11.8-amd64.exe

步骤 2：安装 Python
  1. 双击下载的安装程序
  2. ⚠️ 重要：勾选 "Add Python to PATH"（添加到环境变量）
  3. 点击 "Install Now"
  4. 等待安装完成

步骤 3：验证安装
  1. 按 Win + R 键
  2. 输入 cmd 回车
  3. 输入：python --version
  4. 如果显示版本号（如 Python 3.11.8），说明安装成功

步骤 4：重新打包
  关闭所有命令行窗口，双击 build.bat

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
方法二：使用 Microsoft Store 安装（简单）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

步骤 1：打开 Microsoft Store
  按 Win 键，搜索 "Microsoft Store"

步骤 2：搜索 Python
  在 Store 中搜索 "Python 3.11"

步骤 3：安装
  点击"获取"或"安装"按钮

步骤 4：验证
  打开命令提示符，输入：python --version

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
方法三：手动添加环境变量（如果已安装但未识别）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

步骤 1：找到 Python 安装路径
  常见位置：
  • C:\Users\你的用户名\AppData\Local\Programs\Python\Python311
  • C:\Python311
  • C:\Program Files\Python311

步骤 2：添加到环境变量
  1. 右键"此电脑" → "属性"
  2. 点击"高级系统设置"
  3. 点击"环境变量"
  4. 在"系统变量"中找到"Path"，双击
  5. 点击"新建"，添加 Python 路径
  6. 再添加 Python\Scripts 路径
  7. 点击"确定"保存

步骤 3：重启命令行
  关闭所有命令行窗口，重新打开测试

常见问题

Q: 输入 python 打开了 Microsoft Store？
A: 说明 Python 未正确安装，请按方法一或方法二安装

Q: 提示 'python' 不是内部或外部命令？
A: Python 未添加到环境变量，请按方法三操作

Q: 安装后还是提示未找到？
A: 需要重启命令行窗口，或重启电脑

Q: 需要安装哪个版本？
A: 推荐 Python 3.11.x 或 3.10.x（3.10 以上都可以）

Q: 32位还是64位？
A: 如果你的系统是64位（大部分现代电脑），选择 64位版本
